# Changelog

All notable changes to TypeRank will be documented in this file.

## [0.1.0] - 2026-03-02

### 🎉 Initial Release

#### Features
- **Typing Test** — Monkeytype-inspired typing test with real-time WPM and accuracy
- **3 Word Modes** — Common English words, programming keywords, and quotes
- **4 Timer Durations** — 15s, 30s, 60s, and 120s
- **WPM-over-time chart** — Visual graph of your performance throughout the test
- **6 Rank Tiers** — Bronze, Silver, Gold, Diamond, Master, Legendary
- **8 Achievement Badges** — First Strike, On Fire, Sharpshooter, Speed Demon, Summit, Consistent, Anti-Cheat Verified, Daily Grinder
- **Status Bar Integration** — Live WPM and rank badge in the VS Code status bar
- **Sidebar Panel** — Profile card, personal leaderboard (top 10), badge showcase
- **Theme Support** — Automatic adaptation to dark, light, and high-contrast themes

#### Anti-Cheat
- Paste detection and blocking
- Keystroke interval analysis
- Key hold duration checking
- Burst speed detection
- Focus loss monitoring
- Robotic rhythm detection

#### Commands
- `TypeRank: Start Typing Test` — Opens the typing test panel
- `TypeRank: Show Profile` — Focuses the sidebar
- `TypeRank: Reset All Data` — Clears all history and badges
